# reimagined-fishstick

## Running the application

1. Ensure that Node >= 20 is installed
2. Navigate to the `snurrio` folder
3. Copy the `.env.example` file to `.env.local` and add your OMDb API key
4. Run `npm install`
5. Run `npm run dev`
6. The app should now be running at `http://localhost:3000`
